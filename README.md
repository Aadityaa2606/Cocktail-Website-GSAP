# GSAP Animations Showcase

A showcase project demonstrating various GSAP (GreenSock Animation Platform) animation techniques in a React + TypeScript + Vite application.

## Features

This project showcases multiple GSAP animation techniques including:

- **Scroll Animations** - Scroll-triggered animations that respond to user scrolling
- **Transform Animations** - Element transformations including scale, rotate, and position changes
- **Letter Animations** - Character-by-character text animations
- **And more** - Various other GSAP animation patterns and effects

## Tech Stack

- **React** - UI library
- **TypeScript** - Type safety
- **Vite** - Fast build tool and dev server
- **GSAP** - Professional-grade animation library

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
